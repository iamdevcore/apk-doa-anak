ğŸ”§ Teknologi & Database

Aplikasi Doa Anak-Anak dikembangkan menggunakan teknologi modern untuk memastikan performa yang stabil, tampilan ramah anak, serta kemudahan pengembangan ke depannya.

Frontend: Flutter & Dart

Backend / Database: Firebase (Cloud Firestore)

Local Storage: SharedPreferences

Audio Engine: AudioPlayers

Notifikasi: Flutter Local Notifications

UI Standard: Material Design 3

Firebase digunakan untuk menyimpan data doa secara real-time, sementara progres belajar, favorit, badge, dan target harian disimpan secara lokal agar aplikasi tetap dapat digunakan tanpa koneksi internet.

ğŸš€ Rencana Pengembangan Fitur (Masa Depan)

Aplikasi Doa Anak-Anak direncanakan untuk terus dikembangkan dengan penambahan fitur yang mendukung proses pembelajaran anak secara lebih optimal, antara lain:

Personalisasi target doa harian sesuai usia anak

Sistem badge & reward yang lebih variatif

Statistik perkembangan anak per minggu/bulan

Mode pengawasan orang tua (Parent Control)

Sinkronisasi akun agar data dapat dipindahkan antar perangkat

Pengembangan ini bertujuan untuk meningkatkan konsistensi kebiasaan berdoa, motivasi anak, serta keterlibatan orang tua dalam proses pembelajaran.
